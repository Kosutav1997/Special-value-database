Dedekind Eta Function

The Dedekind eta function is defined for τ in the upper half-plane by

η(τ) = q1/24 ∏n≥1 (1 − qn),   q = e2πiτ.

Modular j-Invariant

The modular j-invariant is a Hauptmodul for SL₂(ℤ), classifying elliptic curves over ℂ up to isomorphism. At CM points, j(τ) is an algebraic integer generating the Hilbert class field of the corresponding imaginary quadratic field.
